## Prerequisites

Before you can run or develop this project, you need to install the following system-level dependencies. These dependencies are required to build Python packages that rely on C extensions or specific libraries (like SQLite).

### 1. Install System Dependencies

For systems that use `apt` (e.g., Ubuntu/Debian), you need to install the following system package:

```bash
sudo apt-get update
sudo apt-get install libsqlite3-dev
pip install -r requirements.txt
```
